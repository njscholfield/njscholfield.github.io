Node

These are some demo apps that I built with Node.js and hosted on Heroku.

Login Demo

I wanted to make a sample web app with authentication so I created this demo. So far this application allows for account registration, login, password change, account deletion, and password resetting. It doesn't really have any practical uses, but I learned a lot about Express.js, MongoDB, Bcrypt, and sessions in Express.

FHIR Demo

This is a demo app that accesses demo patient records from the Cerner FHIR API using a test EMR database supplied by Cerner for development. I wanted to try using a RESTful API to make something somewhat interesting so I make this demo. Using this app you can enter a patient number and pick the type of information you would like to see about that patient. Not all of the categories have any results for all patients so it might be hard to find something interesting.

Form Demo

This is a pretty simple app that just uses a form to input data, stores it in a PostgreSQL database, and displays that data another page. This is one of the first apps I made using Node and was also one of the first apps I made that incorperated a database.
